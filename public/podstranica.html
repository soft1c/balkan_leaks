Halid Bešlić

Amaterski se počeo baviti glazbom, odnosno pjevanjem još kao učenik osnovne škole, kroz razna kulturno umjetnička društva, a najduže vremena je proveo u KUD “Zijo Dizdarević”. To je potrajalo sve do odsluženja vojne obveze u bivšoj SFRJ. Nakon povratka iz vojske svoju glazbenu karijeru je više aktivirao kroz nastupe u poznatim sarajevskim i drugim restoranima, što je potrajalo nekih 5-6 godina. Prvu singl-ploču snimio je 1979. godine od kada počinje njegov uzlazni put uspjeha i popularnosti na bosanskoj, odnosno balkanskoj glazbenoj sceni. Dodatni filteri

Search operators can help narrow your search with a lot of flexibility. Some operators can also allow you to be more vague in your search parameters - for instance by specifying that all matching documents must contain a certain number of some given words.

The operators must be typed into the usual search box, and most of them can be combined with other search operators.

You can read detailed description of most of the operators and how to use them on Search WikiLeaks page. Here are few most common examples.

Search operators

There is a number of fields you can use to filter documents in this set.

@file_type find files based on their type, for example "@file_type pdf" will find PDF files

@file_name will search for given filename

@file_path will search with full path

@file_folder will search for files contained within named folder
